# Создание планет

Этот раздел справки содержит информацию о создании своих планет для игры Lonely Space.

### Важно

Файл с планетами находится здесь: base/game/res/**planets.json**

Он содержит в себе список всех доступных планет для игрока.

### Как сделать свою?

Давайте рассмотрим это на примере планеты Земли.

```
  {
    "id": 3,
    "name": "Земля",
    "description": "Когда-то зелёная и голубая планета, полная жизни, но в нынешнем состоянии она пережила некую катастрофу, оставившую её неопределённой и пугающей.\nОстатки цивилизаций покрыты пылью и зарослями. Климат сильно нестабилен: океаны отступили или мутировали в новые опасные формы жизни. Будущее этой планеты окутано туманом неизвестности.",
    "type": 4,
    "danger": 3,
    "eta": 60,
    "temperature": -10
  },
```
- **id** - Уникальный номер планеты. Я просто вписываю номер планеты по порядку в списке.
- **name** - Название планеты.
- **description** - Описание планеты. Оно содержит информацию об атмосфере планеты, а также возможную историю этой планеты. ВАЖНО: Описание не должно быть очень длинным, иначе будут проблемы с отображением в игре.
- **type** - Тип планеты. 

> Записка от пришельцев, типы планет:
> 
> 0 - Газовый гигант 
> 
> 1 - Каменистая 
> 
> 2	- Ледяная 
> 
> 3	- Вулканическая 
> 
> 4	- Океаническая 
> 
> 5	- Пустынная 
> 
> 6	- Токсичная

- **danger** - Уровень опасности планеты. От 0 до 9. От этого зависят события на планете, а также возможные награды за её посещение. Более опасная планета - более высокие награды, но выше шанс погибнуть. 

- **eta** - Среднее время полёта до этой планеты (в секундах)

- **temperature** - Среднее значение температуры (в Цельсиях) на этой планете.

### Шаблон
```
  {
    "id": [ЧИСЛО],
    "name": "[НАЗВАНИЕ]",

    "eta": [ЧИСЛО],

  },
```
