# Создание планет

Этот раздел справки содержит информацию о создании своих планет для игры Lonely Space.

### Планеты сообщества

**Планеты сообщества** - это выдуманные планеты, которые создали обычные игроки со своими гениальными идеями. Как насчёт
планеты из ананасов?

**Официальные планеты** - это планеты, которые доступны в базовой комплектации игры, проверяются мною, и содержат
существующие планеты. Несмотря на то, что это официальный список, Ваши планеты также могут оказаться здесь.
Для этого нужно выполнить 2 условия:

1. Планета существует в реальности
2. Параметры планеты настроены с учетом реальной информации о планете.

### Важно

Файл с официальными планетами находится здесь:

    base/game/res/planets.json

Файл с планетами сообщества находится здесь:

    custom/custom_planets.json

### Как сделать свою?

Давайте рассмотрим это на примере планеты Земли.

```
  {
    "id": 3,
    "name": "Земля",
    "description": "Когда-то зелёная и голубая планета, полная жизни, но в нынешнем состоянии она пережила некую катастрофу, оставившую её неопределённой и пугающей.\nОстатки цивилизаций покрыты пылью и зарослями. Климат сильно нестабилен: океаны отступили или мутировали в новые опасные формы жизни. Будущее этой планеты окутано туманом неизвестности.",
    "type": 4,
    "danger": 3,
    "eta": 60,
    "temperature": -10,
    "events": [
      {
        "name": "Руины мегаполиса",
        "description": "Экспедиция обнаружила остатки древнего мегаполиса с уцелевшими хранилищами.",
        "commands": [
          "resources add 35 15"
        ],
        "prob": 0.08,
        "color": "green"
      },
      {
        "name": "Нестабильная биосфера",
        "description": "Мутанты атаковали наш корабль!",
        "commands": [
          "crew remove 15 5"
        ],
        "prob": 0.04,
        "color": "red"
      }
    ]
  },
```

- **id** (число) - Уникальный номер планеты. От 256 до 4096. Это число не должно повторяться с ID других планет.
- **name** (строка) - Название планеты.
- **description** (строка) - Описание планеты. Оно содержит информацию об атмосфере, а также возможную историю этой
  планеты. ВАЖНО: Описание не должно быть очень длинным, иначе будут проблемы с отображением в игре.
- **type** (число) - Тип планеты. От типа планета зависят особые случайные события, которые могут произойти на ней.

> Записка от пришельцев, типы планет:
>
> 0 - Газовый гигант
>
> 1 - Каменистая
>
> 2 - Ледяная
>
> 3 - Вулканическая
>
> 4 - Океаническая
>
> 5 - Пустынная
>
> 6 - Токсичная

- **danger** (число) - Уровень опасности планеты. От 0 до 9. От этого зависят события на планете, а также возможные
  награды за её посещение. Более опасная планета - более высокие награды, но выше шанс погибнуть.

- **eta** (число) - Среднее время полёта до этой планеты (в секундах). Рекомендуются значения в пределах от 45 до 120. В
  любом случае, я не ограничиваю вас :)

- **temperature** (число) - Среднее значение температуры (в Цельсиях) на этой планете.

- **events** - События на планете.

### События

Вы можете создавать свои собственные события, которые могут происходить на планетах.

Подробнее про создание событий вы можете узнать [здесь](database/CREATING-EVENTS.md#события).

События на планетах вписываются в блок **events**. Смотрите на пример выше.

### Шаблон Планеты

```
  {
    "id": ЧИСЛО,
    "name": "НАЗВАНИЕ",
    "description": "ОПИСАНИЕ",
    "type": ЧИСЛО,
    "danger": ЧИСЛО,
    "eta": ЧИСЛО,
    "temperature": ЧИСЛО,
    "events": [
    ]
    "author": "ТЕКСТ"
  },
```

> Обратите внимание!
>
> Появилось значение **author**
>
> В нём вы можете указать себя, как автора данной планеты.
>
> Пример:
>
> `"author": "pxffd"`
