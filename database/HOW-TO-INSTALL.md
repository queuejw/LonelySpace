# Как скачать игру?

## Подготовка перед скачиванием игры

1. Убедитесь, что у вас установлен Python последней версии.

   ###### Не знаете, установлен ли у Вас Python или не знаете какой он версии? Перейдите к разделу [Помощь](#помощь).
2. Если Python не установлен, загрузите его здесь:

    - Если у вас Windows 8.1 или выше:

        - Перейдите [по этой ссылке](https://www.python.org/downloads/release/python-3142/).
        - Прокрутите вниз
        - Если у вас **64**-разрядная операционная система, нажмите на **Windows installer (64-bit)**

        - Если у вас **32**-разрядная операционная система, нажмите на **Windows installer (32-bit)**

      ###### Не знаете, какая разрядность у Вашей ОС? Перейдите к разделу [Помощь](#помощь).

        - Запустите скачанный файл и установите Python, следуя инструкциям Установщика.


- Если у вас Windows 8 и ниже:

    - Перейдите [по этой ссылке](https://github.com/adang1345/PythonVista/tree/master/3.13.10).
    - Если у вас **64**-разрядная операционная система, загрузите файл ...**-amd64-full.exe**

    - Если у вас **32**-разрядная операционная система, загрузите файл ...**-full.exe**
  ###### Не знаете, какая разрядность у Вашей ОС? Перейдите к разделу [Помощь](#помощь).
    - Запустите скачанный файл и установите Python, следуя инструкциям Установщика.


- Если у вас Linux:
    - Для Arch-based дистрибутивов обратитесь
      к [документации](https://wiki.archlinux.org/title/Python_(%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9)) Arch Linux.

    - Для Debian-based дистрибутивов обратитесь либо к [документации Debian](https://wiki.debian.org/Python), либо
      документации Вашего дистрибутива.

    - Стоит добавить, что Python обычно предустановлен на дистрибутивах Linux. Обратитесь к
      разделу [Помощь](#помощь), чтобы узнать установленную версию Python.

## Скачивание игры

Чтобы загрузить последнюю версию игры, перейдите по [этой](https://github.com/queuejw/lonelyspace/releases/latest)
ссылке.

Затем, найдите и загрузите файл **lonely-space-x.x.x.zip**

Если вы хотите помочь мне, можете скачать последнюю, нестабильную версию игры
по [этой](https://github.com/queuejw/LonelySpace/archive/refs/heads/master.zip) ссылке. Она содержит самые последние
изменения, но может оказаться нестабильной. Делитесь своим мнение об этой версии, чтобы помочь сделать игру лучше для
всех.

**После загрузки ZIP файла, распакуйте его в любом удобном месте при помощи архиватора.**

## Установка и запуск игры

После распаковки игры, Вам потребуется установить требуемые для игры библиотеки.

Для этого:

1. Откройте папку с игрой
2. Запустите **install_requirements.bat** (если у Вас Windows) или **install_requirements.sh** (если у Вас Windows)

После установки библиотек (вы увидите сообщение об успешной установке) можно начать игру.

Запустите **launch_lonely_space.bat** (если у Вас Windows) или **launch_lonely_space.sh** (если у Вас Linux)

# Помощь

### Почему не скачивается Python?

Проверьте подключение к интернету.

Если проблема только с python.org, загрузите Установщик по [этой](https://pythonlinks.python.jp/en/index.html) ссылке.

### Какая у меня версия Python и установлен ли он вообще?

1. Откройте Windows PowerShell (Windows) или Терминал (Linux)
2. Введите команду **python --version**

Эта команда должна вывести на экран текущую версию Python
> Пример:
>
> Python 3.14.2

Если Вы **не** увидели что-то похожее, возможно у Вас **не установлен Python**.

Для стабильной работы игры требуется Python версии 3.13 и новее.

### Как определить разрядность моей операционной системы?

- [Инструкция для Windows](https://ru.wikihow.com/%D0%BE%D0%BF%D1%80%D0%B5%D0%B4%D0%B5%D0%BB%D0%B8%D1%82%D1%8C-%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%8E-Windows-%E2%80%93-32-%D0%B8%D0%BB%D0%B8-64-%D1%80%D0%B0%D0%B7%D1%80%D1%8F%D0%B4%D0%BD%D0%B0%D1%8F)
- Linux:
    - Современные дистрибутивы Linux обычно 64-разрядные, но если Вы не уверены, проверьте, используя инструкцию ниже.
    -
        - Введите команду **uname -m** в терминал.
    -
        - Если в строке ответа упоминается x86_64, это означает, что система 64-разрядная.
    -
        - Если в строке ответа упоминается i386, i486, i586 или i686, это означает, что система 32-разрядная.
