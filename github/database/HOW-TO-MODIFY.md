## Основное

- **base**/ - база игры. Здесь находится весь основной код игры.


- base/**core**/ - здесь находится весь внутренний функционал, который не связан с игрой. Например, чтение файлов или
  работа с консолью.

-
    - **components.py** - связующее звено между различными частям игры. Содержит переменные, которые создаются при
      запуске игры и используются во всём коде игры.

-
    - **console.py** - всё, что связано с консолью (не игры). Например, очистка экрана.

-
    - **constants.py** - значения, которые не изменяются во время игры. Например, названия директорий, игры и т.д.

-
    - **settings.py** - класс настроек игры.

-
    -
        - **json_manager.py** - работа с json файлами (чтение и запись)

-
    -
        - **planet_manager.py** - нужен для загрузки json файла с планетами из директории *res*.

-
    -
        - **txt_loader.py** - загружает txt "рисунок" корабля игрока из директории *res*.


- base/**game** - здесь находится всё, что как-либо связано с игрой

-
    - base/game/**classes**/ - код игры находится здесь.

-
    -
        - **engine.py** - движок игры. Обеспечивает получение ввода и обновления экрана.

-
    -
        - **game.py** - весь код игры здесь (почти). Здесь же происходит генерация событий, обработка полётов, генерация
          текста интерфейса и т.д.

-
    -
        - **planet.py** - класс планеты.

-
    -
        - **ship.py** - класс корабля игрока.


-
    - base/game/classes/**ui** - здесь находится код интерфейса (экрана) игры.

-
    -
        - **base/screen.py** - базовый объект экрана, на основе которого созданы следующие.

> Здесь происходит обработка ввода, полученного движком игры.

-
    -
        - **main_menu.py** - Главное меню игры, которое вызывается при запуске.

-
    -
        - **game_screen.py** - Экран игры.


-
    - base/game/**res**/ - ресурсы игры (планеты, корабль, аудио)

-
    -
        - **planets.json** - список планет.

-
    -
        - **ship.txt** - рисунок корабля
