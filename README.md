# Lonely Space

Добро пожаловать в Пустой Космос. Это простая текстовая игра про выживание в открытом космосе после жуткой катастрофы на
Земле. Будьте осторожны, ведь шансы на выживание уменьшаются с каждой секундой! Может быть, кто-нибудь
спасёт вас?

## Системные требования

- Компьютер (ао возможности)
- Python 3.6 и новее.
- Доступ к интернету (для скачивания игры)

## Подготовка к запуску игры.

1. Загрузите Python с официального сайта. Этот шаг нужен для Windows, поскольку Python обычно предустановлен на
   дистрибутивах Linux
    - https://www.python.org/
    - Если у вас Linux и у вас не установлен Python, установите его самостоятельно, пользуясь инструкциями из Интернета
2. Установите необходимые библиотеки из <b>requirements.txt</b>
    - colorama
    - keyboard
3. Запустите при помощи Python файл <b>main.py</b>
    - python .\main.py
